# memstim-hh
MEMory STIMulation, a dynamical computational model of hippocampal theta-nested gamma oscillations that interfaces Kuramoto oscillators with Hodgkin-Huxley neurons and exhibits theta phase reset.

PhD Thesis Project.

Setup instructions can be found in /setup-env/instructions.txt

This code is used in the following publication:
"Vardalakis Nikolaos, Aussel Amélie, Rougier Nicolas P., Wagner Fabien B. (2023) A dynamical computational model of theta generation in hippocampal circuits to study theta-gamma oscillations during neurostimulation eLife 12:RP87356
https://doi.org/10.7554/eLife.87356.2
